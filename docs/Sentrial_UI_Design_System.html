1. Fonts & Typography

Two-font system: Sentrial uses Space Grotesk (geometric sans-serif) for the product UI and Playfair Display (serif) for marketing/hero headings. Code uses JetBrains Mono.

8. Design Principles & Notes

What makes the Sentrial UI feel "smooth"

1. Ultra-dark, true-black base. Using #000000 as the root background creates extreme contrast. Cards at #0C0E13 (just 6% lightness) float subtly above — the 6% difference is barely visible but creates depth without harsh edges.

2. Extremely restrained border system. Only ONE border color (#1E1E1F) is used everywhere. This consistency makes the entire interface feel unified. Borders are 1px solid — never thicker, never dashed.

3. Compact, dense typography. Text sizes are notably small (10-14px for most UI elements). This creates a data-dense, professional feel. The nav items at 13px and table headers at 10px uppercase give it a terminal/engineer aesthetic.

4. Minimal color palette. Almost everything is grayscale. Color is reserved for: green (success/progress), red (errors), orange (brand accent/PRO badge). This makes colored elements instantly noticeable.

5. Two-font personality. Space Grotesk (geometric, modern) for the product creates a technical feel. Playfair Display (serif, editorial) for marketing creates premium contrast. They never mix within the same context.

6. Glassmorphism-ready. The CSS variables include glass effects (--glass-bg, --glass-border) suggesting backdrop-filter usage for overlays and popovers.

7. Micro-interactions. All interactive elements use 150ms ease transitions. Hover states add barely-visible white overlays (rgba(255,255,255,0.05)) rather than color changes — keeping the dark aesthetic intact.
